Sage 50 Network Installation Manager Instructions

Sage 50 has been installed on your server. Next, you need to install it on each workstation.

 

 Step 1 Get these instructions to the other computers on your network

  •  Email this file to an account that you can access from the other computer on your network.  
Or
  •  Save this file to a USB drive and take it to each workstation.
Or
  •  Print these instructions and take them to each workstation.

  Once you are at the workstation, open the Sage 50 - Network Installation Manager
 Instructions.htm file.

 Step 2 Install Sage 50 on each workstation

  From each workstation, click the following link to start the installation:

  \\WINSERVER\PeachtreeInstaller2022\ClientInstall.bat

  Note: If the link above doesn't work, copy/paste or type it in Start>Run or in Windows
 Explorer on each workstation.

  Tips:

  • You can install on multiple workstations at the same time. Simply start the installation on one workstation and then move on to the next one.
  • If you have any problems using the network installation manager, you can still install Sage 50 on each workstation using the DVD or download files.

Network Requirements:

We recommend that you have at least a 100 megabit per second network and an active directory domain to use the network installation manager. The network installation manager will not work with wireless networks.

Note: If you do not have an active directory domain, you can still use the network installation manager, but it may have problems trying to automatically connect.
